Project Overview

ShareDrop is a lightweight, fast, and secure web application that allows users to instantly share files and text across devices. With a simple drag-and-drop interface and QR code support, it enables seamless cross-platform sharing without requiring login or complex setup.

Built with Laravel framework, ShareDrop ensures fast, secure, and hassle-free file transfers. It's perfect for quick sharing between phone and desktop, making it an essential tool for everyday file transfers without the need for email attachments or cloud storage accounts.

The application generates unique shareable links and QR codes for each upload, allowing recipients to download files directly without any authentication requirements.

Technical Implementation

ShareDrop is built using Laravel 10 framework with a MySQL database for storing file metadata. The frontend utilizes Bootstrap 5 for responsive design and jQuery for dynamic interactions including drag-and-drop functionality.
